# CidadãoAtivo – Plataforma Completa

Backend (Flask) + Frontend (Node.js)

Plataforma de engajamento cívico que conecta cidadãos e vereadores para envio, aprovação e acompanhamento de demandas.

## Instruções de Frontend

1. **Instalar dependências:**

Execute o comando npm install dentro da pasta do frontend.

2. **Rodar o projeto:**

Execute npm run dev para iniciar o ambiente de desenvolvimento.

A interface abrirá normalmente em http://localhost:5173

## Instruções de Backend

1. **Instalar dependências::**
   
cd backend
Instale tudo usando o comando pip install -r requirements.txt

2. **Inicializar o banco de dados::**

Execute o arquivo seed_data.py para criar a base com os dados iniciais.

3. **Rodar o servidor:**

Execute: python app.py

O servidor ficará disponível em: http://localhost:5000
